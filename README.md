# 绿灯行 - 盲人过马路红绿灯识别助手

> 用手机AI实时识别红绿灯，帮助盲人安全过马路

## 项目简介

刷短视频看到盲人过马路的画面，触动了我。用 TRAE SOLO 从一个念头出发，2小时完成了市场调研、产品设计、可交互原型、四轮代码评审、真实AI接入的全流程。

```

```

## 快速开始

### 1. 安装依赖

```bash
pip install flask ultralytics opencv-python
```

### 2. 启动服务器

```bash
python server.py
```

### 3. 打开页面

- 电脑浏览器：http://localhost:5000
- 手机浏览器：http://电脑IP:5000（需同一WiFi）

### 4. 使用方法

1. 打开页面后，先看3步引导
2. 点击"开始使用"进入主页面
3. 允许摄像头权限
4. **按住**中央大按钮，对准红绿灯
5. AI会实时识别并语音播报结果

## 技术栈

| 组件 | 技术 |
|------|------|
| 前端 | HTML5 + CSS3 + JavaScript |
| 摄像头 | getUserMedia API |
| 语音播报 | Web Speech API |
| 后端 | Python + Flask |
| AI模型 | YOLOv8n（目标检测） |
| 颜色判断 | OpenCV + HSV颜色空间分析 |

## 产品特点

- **极简交互**：一个按钮，按住识别，松开停止
- **无障碍适配**：大字号、高对比度、语音播报、振动反馈
- **置信度策略**：不确定时提示用户确认，避免误报
- **信号变化检测**：红绿灯变色时立即告警
