# 推送到GitHub的步骤

Git仓库已初始化并提交，只需以下步骤：

## 步骤

```bash
# 1. 在GitHub上创建一个新仓库（不要勾选README）
#    https://github.com/new
#    仓库名：green-light-go
#    设为 Public
#    不要勾选任何初始化选项

# 2. 添加远程仓库并推送
cd /你的项目路径/green-light-go
git remote add origin https://github.com/jinqianfei/green-light-go.git
git push -u origin main
```

## 如果提示需要认证

GitHub 现在需要使用 Personal Access Token 或 SSH 密钥：

1. **Personal Access Token 方式**：
   - 访问 https://github.com/settings/tokens
   - 生成新的 token，勾选 `repo` 权限
   - 推送时用 token 代替密码

2. **SSH 方式**：
   ```bash
   git remote set-url origin git@github.com:jinqianfei/green-light-go.git
   git push -u origin main
   ```
