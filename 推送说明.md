# 推送到GitHub的步骤

## 方法一：命令行推送（推荐）

```bash
# 1. 在GitHub上创建一个新仓库（不要勾选README）
#    https://github.com/new
#    仓库名：green-light-go
#    设为Public

# 2. 在项目文件夹中打开终端，执行以下命令：

git init
git add .
git commit -m "feat: 盲人过马路红绿灯识别助手 - 初始版本"
git branch -M main
git remote add origin https://github.com/jinqianfei/green-light-go.git
git push -u origin main
```

## 方法二：GitHub网页上传

1. 打开 https://github.com/new
2. 仓库名填 `green-light-go`，设为 Public
3. 不要勾选任何初始化选项
4. 点 Create repository
5. 点 "uploading an existing file"
6. 把项目文件夹里的文件拖进去
7. 点 Commit changes
