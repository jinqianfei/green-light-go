# 绿灯行 - 后端部署指南

## 方案一：Render 部署（推荐，免费）

Render 免费套餐支持 Python Web Service，自带 512MB 内存，足够跑 YOLOv8n。

### 一键部署（推荐）

项目已包含 `render.yaml`，你只需：

1. 将仓库推送到 GitHub
2. 登录 Render
3. 选择 "New" → "Web Service"
4. 选择 "Deploy from GitHub"
5. 选中当前仓库和 `main` 分支
6. Render 会自动识别 `render.yaml` 并使用以下配置：
   - **Name**: `green-light-go`
   - **Environment**: Python
   - **Build Command**: `pip install -r requirements.txt`
   - **Start Command**: `gunicorn --bind 0.0.0.0:$PORT --timeout 120 --workers 1 --threads 4 server:app`
   - **Environment Variable**: `YOLO_MODEL_PATH=yolov8n.pt`

### 手动部署（备用）

1. **注册 Render**
   - 打开 https://render.com ，用 GitHub 账号登录

2. **创建 Web Service**
   - 点 "New" → "Web Service"
   - 连接你的 GitHub 仓库
   - 配置如下：
     - **Name**: `green-light-go`
     - **Runtime**: Python 3
     - **Build Command**: `pip install -r requirements.txt`
     - **Start Command**: `gunicorn --bind 0.0.0.0:$PORT --timeout 120 --workers 1 --threads 4 server:app`
     - **Instance Type**: Free

3. **上传 YOLOv8 模型文件**
   - yolov8n.pt 大约 6MB，Render 免费套餐会从 GitHub 拉取
   - 把 `yolov8n.pt` 提交到 GitHub 仓库（.gitignore 里已移除排除）
   - 或者：在 Render 的 Environment 里设置 `YOLO_MODEL_PATH` 为模型下载URL

4. **等待部署完成**
   - Render 会自动安装依赖、启动服务
   - 首次启动需要下载模型，约 1-2 分钟
   - 部署成功后会得到一个 URL，如 `https://green-light-go.onrender.com`

5. **前端配置**
   - 如果前后端都部署在同一个服务上（同域名部署），不需要改 `API_BASE`，保持 `var API_BASE = '';` 即可
   - 如果前端单独部署（如你现有的 serveousercontent 链接），需要修改 `绿灯行真实版.html` 中的：
     ```javascript
     var API_BASE = 'https://green-light-go.onrender.com';
     ```

### 注意事项
- Render 免费套餐会在 15 分钟无流量后休眠，首次请求需要等 30-60 秒冷启动
- 冷启动时模型加载较慢，建议设置 `--timeout 120`

## 方案二：Railway 部署

Railway 免费额度 $5/月，适合小项目。

1. 打开 https://railway.app ，用 GitHub 登录
2. "New Project" → "Deploy from GitHub repo"
3. 选择 `green-light-go` 仓库
4. Railway 会自动检测 `requirements.txt` 和 `Procfile`
5. 添加环境变量（如需要）：`YOLO_MODEL_PATH=yolov8n.pt`
6. 部署完成后得到 URL

## 方案三：本地运行（已有电脑 + 手机同WiFi）

```bash
# 1. 安装依赖
pip install flask flask-cors ultralytics opencv-python-headless --break-system-packages

# 2. 下载模型（首次运行会自动下载）
# 如果已有 yolov8n.pt 放在同一目录即可

# 3. 启动服务
python server.py

# 4. 手机浏览器打开
# http://电脑局域网IP:5001
# （查看IP：ifconfig 或 ipconfig）
```

## 文件说明

| 文件 | 用途 |
|------|------|
| `server.py` | Flask 后端，YOLOv8 红绿灯识别 |
| `绿灯行真实版.html` | 前端页面（摄像头 + 语音播报） |
| `requirements.txt` | Python 依赖列表 |
| `Procfile` | 云平台启动命令 |
| `runtime.txt` | Python 版本（Render 用） |
| `yolov8n.pt` | YOLOv8 nano 模型文件（需提交到仓库） |

## .gitignore 注意

当前 .gitignore 排除了 `*.pt` 文件。如果要部署到云端，**需要把 yolov8n.pt 提交到仓库**：

```bash
# 临时取消排除
git add -f yolov8n.pt
git commit -m "chore: add yolov8n model for deployment"
git push
```

或者修改 .gitignore，把 `*.pt` 那行删掉。
